# 💙 FeiTask

A modern and responsive task management web application built with HTML, CSS, and JavaScript.

## ✨ Features

- Add new tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Progress bar
- Dark mode
- Local Storage
- Responsive design

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
